Ferramentas:

Existem diversas ferramentas que podemos utilizar para criar experiências em realidade virtual com o A-Frame. Segue algumas:

Glitch - Editor de código on-line indicado pela documentação oficial do A-Frame
CodePen - Editor de código on-line e um ambiente de aprendizado de código aberto
Visual Studio Code - Editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS
